# Herbert

A Claude-powered, voice-enabled AI assistant device. Runs on a Raspberry Pi 5 (production) and macOS (development) from the same codebase.

> Named after Rodney Brooks' 1988 MIT soda-can-collecting robot, itself named for AI pioneer Herbert Simon.

Herbert is a personal tinker project. The device existing is the point — features serve a retro-futurist, character-forward vibe.

## Status
